# My Terraform Cheat Sheet

## Graph

```sh
terramform graph
```

## Console

```sh
terramform console
```

## Target

```sh
terramform apply -target=<resource-type>.<resrouce-name>
```

## Refersh

```sh
terramform apply -refresh=<boolean>
```
